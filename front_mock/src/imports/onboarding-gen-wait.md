
• 了解です。
  「生成待ち6ページ（1分ループ）」の詳細計画を、実装できる粒度で整理します。

  ## 全体仕様

  - ループ長: 60秒
  - ページ数: 6ページ（1ページ 10秒）
  - 表示形式: 静止ドット絵 + 白ドット文字2〜3行
  - 進行: 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 1...
  - 生成完了時: ループ停止して「会話をはじめる」ボタン表示

  ———

  ## ページ別詳細（ドット絵 + テキスト）

  ### Page 1（0-10s）: 接続開始

  - ドット絵:
      - 暗い背景
      - 中央奥に小さな白いゲート
      - ゲート前に小さいシルエット（未完成）
  - テキスト:
      - AGENT WORLD 接続中...
      - 君のクローン生成を開始した
  - 演出:
      - ゲートの外周だけゆっくり点滅

  ### Page 2（10-20s）: 声紋同期

  - ドット絵:
      - 波形っぽい白ラインが左右に走る
      - シルエット周辺に小粒子
  - テキスト:
      - VOICE SYNC 進行中
      - 声の粒をコアに編み込んでいる
  - 演出:
      - 波形ラインが左右にスキャン

  ### Page 3（20-30s）: 人格マップ構築

  - ドット絵:
      - マス目（グリッド）に白い点が増える
      - 点同士を細線で接続
  - テキスト:
      - PERSONA MAP 構築中
      - 言葉の癖と判断軸を固定している
  - 演出:
      - 点が順番に点灯

  ### Page 4（30-40s）: 顔アバター生成

  - ドット絵:
      - 顔パーツの輪郭（目・口・輪郭）が段階表示
      - 2等身体テンプレの首上に重なる構図
  - テキスト:
      - AVATAR FACE COMPILE
      - 君の面影を2等身ボディへ接続
  - 演出:
      - 輪郭が1本ずつ描かれる風

  ### Page 5（40-50s）: 最終リンク

  - ドット絵:
      - ゲートが大きく明るくなる
      - シルエットに白縁が入る
  - テキスト:
      - FINAL LINK...
      - 起動まで、あと少し
  - 演出:
      - 画面全体の輝度が少し上がる

  ### Page 6（50-60s）: 準備完了待機

  - ドット絵:
      - 2等身アバターの静止立ち絵（薄く）
      - ゲートは安定発光
  - テキスト:
      - READY
      - 生成完了で進行ボタンが表示される
  - 演出:
      - 最小限の呼吸感（発光だけ）

  ———

  ## 音響計画（BGM/SE）

  - BGM:
      - 低音アンビエント1曲を全ページ共通でループ
  - ページ切替SE:
      - 10秒ごとに小さな「tick」
  - 生成完了SE:
      - やや明るい短い「confirm」音
  - ルール:
      - 音量は控えめ（会話体験を邪魔しない）

  ———

  ## 実装ルール（状態管理）

  - 状態:
      - GeneratingAssets 中のみ6ページ再生
  - 20秒経過:
      - 先行チャット開始可能
      - ただし生成未完了ならこの6ページループを維持
  - 完了検知:
      - voice_ready && avatar_ready でループ停止
      - CTAボタン表示
  - 失敗時:
      - 失敗専用オーバーレイ（再試行ボタン）
