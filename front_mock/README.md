# OpenClone - あなたの分身AIエージェント

OpenCloneは、個人向け分身AIサービスのフロントエンドモックアプリケーションです。スマートフォン上で常時開いて横に置いて使える日常チャット型AIエージェントを提供します。

## 🌟 特徴

- **暗闇から誕生する分身**: ドット絵の世界観で統一された没入感のあるオンボーディング体験
- **2等身ドット絵アバター**: AIの応答に連動して動くピクセルアートキャラクター
- **モバイルファースト設計**: iOS SafariとAndroid Chromeに最適化
- **PWA対応**: ホーム画面に追加してネイティブアプリのように使用可能
- **音声インタラクション**: テキストと音声の両方で会話可能

## 🎨 デザインコンセプト

- **テーマ**: 「暗闇から誕生し、世界に現れる分身」
- **カラー**: 黒を基調とした暗闇の世界観（#050505）、白い光の演出
- **タイポグラフィ**: ドット絵フォント（Press Start 2P）と可読性の高い本文フォント（Inter）

## 📱 画面構成

### オンボーディングフロー（5ステージ）

1. **Stage 0: 暗闇の導入** - 遠方に小さな光点、導入メッセージ
2. **Stage 1: 基本情報入力** - 名前と顔写真のアップロード
3. **Stage 2: 音声質問フェーズ** - 5〜8問の音声回答（必須）
4. **Stage 3: 生成待機フェーズ** - 声質学習とアバター生成のシミュレーション
5. **Stage 4: 誕生演出** - 完了メッセージと「会話をはじめる」ボタン

### メイン画面（チャット）

- **上部**: 2等身ドット絵アバター表示領域
- **中央**: 会話ログエリア
- **下部**: テキスト入力欄、音声入力ボタン、送信ボタン

## 🚀 開発環境

- **フレームワーク**: React 18.3 + Vite
- **ルーティング**: React Router v7
- **スタイリング**: Tailwind CSS v4
- **アニメーション**: Motion (Framer Motion)
- **UI コンポーネント**: Radix UI
- **言語**: TypeScript

## 💻 起動方法

### デモ・ハッカソン用（フロントのみ・バックエンド未接続）

**デフォルトでモックモード**で動作します。環境変数なしでそのまま起動すれば、バックエンドに接続せずフロントのみでオンボーディング・チャットを体験できます。

```bash
# 依存関係のインストール
pnpm install   # または npm install

# 開発サーバーの起動
pnpm dev       # または npm run dev

# ビルド（Vercel など）
pnpm build     # または npm run build
```

- ランディングで「はじめる」→ オンボーディング → チャットまで全てモックで動作
- バックエンド接続する場合は `.env` に `VITE_USE_API_MOCK=false` と `VITE_API_BASE_URL` を設定

## 🎯 アバターアクション

アバターは以下のアクションに対応しています：

- `idle`: 待機状態（軽い上下動）
- `thinking`: 考え中（微振動）
- `speaking`: 話している（軽いスケール変化）
- `nod`: うなずき
- `agree`: 同意（うなずき）
- `surprised`: 驚き（上下の大きな動き）
- `emphasis`: 強調（拡大）
