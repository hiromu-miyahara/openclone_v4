# OpenClone v4

個人向け分身AIサービス OpenClone のリポジトリです。

- **GitHub**: [https://github.com/hiromu-miyahara/openclone_v4](https://github.com/hiromu-miyahara/openclone_v4)
- **デプロイ・プッシュ**: 必ず **hiromu-miyahara** アカウントで行ってください。

## 構成

| ディレクトリ | 説明 |
|-------------|------|
| `front_mock/` | フロントエンド（React + Vite）・PWA |
| `backend/` | OpenClone ラッパーAPI |
| `docs/` | 要件・設計・契約 |
| `experiments/` | ピクセルアート・音声検証など |

フロントの起動・ビルドは `front_mock/` を参照してください。
