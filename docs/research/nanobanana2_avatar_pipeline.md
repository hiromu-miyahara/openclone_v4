# Nanobanana2 顔ドット絵パイプライン（MVP）

## 1. 目的
オンボーディングで受け取った顔写真から、2等身アバターに合成可能な透過ドット絵顔画像を安定生成する。

## 2. 採用フロー（固定）
1. 入力写真をアップロードしてCloud Storageへ保存
2. Nanobanana2で顔画像を生成して取得
3. 正方形クロップ + `512x512` 正規化
4. `nearest` 補間で `64x64` へ縮小
5. 16-32色に減色（パレット固定）
6. 単色背景キー抜き + 1px輪郭クリーンアップ
7. 透過PNGを書き出し（`face_64.png`）
8. 確認用拡大画像（`face_128_preview.png`）を出力

## 3. 出力仕様
- 本番用: `face_64.png`（透過PNG, RGBA）
- プレビュー用: `face_128_preview.png`（透過PNG, RGBA）
- ジョブ状態APIでは `asset_urls` と `pipeline_version` を返却する

## 4. 品質チェック（MVP最小）
- 解像度が `64x64` 固定
- 背景が透過されていること（alpha channelあり）
- 色数が上限以内（32色以下）
- 白フチ/穴抜けが目視で許容範囲

## 5. 代替案
Nanobanana2方式が品質・安定性・速度のいずれかで要件未達の場合のみ、A方式（MediaPipe + OpenCV + Pillow）へ切替を検討する。
